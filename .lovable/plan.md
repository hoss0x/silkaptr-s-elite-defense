

# SilKaptr - Elite Cybersecurity Agency Landing Page

## Overview
A stunning, dark-mode landing page for SilKaptr, a specialized cybersecurity company offering AI-augmented penetration testing and red teaming services. The design will convey confidence, secrecy, and elite professionalism.

---

## Visual Design System

### Color Palette
- **Primary Background**: Deep black (#050505) with charcoal grey accents
- **Accent Gradient**: Electric Cyan (#00F0FF) to Neon Purple (#A855F7)
- **Text**: White for headlines, muted grey for body text
- **Cards**: Glassmorphism with subtle borders and blur effects

### Typography
- **Headlines**: Inter (bold, modern Sans-Serif)
- **Body Text**: Inter (regular weight)
- **Code/Technical Subtitles**: Fira Code (monospace) for terminal aesthetic

### Effects & Animations
- Dynamic particle background with network node connections
- Subtle glitch effects on hover for interactive elements
- Fade-in animations on scroll
- Glassmorphism cards with backdrop blur
